## Deploy to Vercel

1. Import this repository in Vercel.
2. Framework Preset: **Other**.
3. Build Command: leave empty.
4. Output Directory: leave empty / root.
5. Deploy.
6. Add the custom domain `khasbim.web.id` in Vercel Project Settings в†’ Domains.
